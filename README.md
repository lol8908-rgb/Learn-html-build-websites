# Learn HTML — Build Your First Website 🌐

A beginner-friendly HTML tutorial — learn how to build your first website from scratch. No experience needed.

---

## What You Will Learn

- What HTML is and how it works
- How to structure a webpage
- Text, headings, and paragraphs
- Links and images
- Lists, tables, and forms
- How to publish a website

---

## Who Is This For?

Complete beginners who want to learn web development.  
No prior coding experience required.
