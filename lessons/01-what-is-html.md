# Lesson 01 — What is HTML?

## What is HTML?

HTML stands for **HyperText Markup Language**.  
It is the foundation of every website on the internet.

HTML tells the browser **what to display** — headings, paragraphs, images, links, and more.

---

## Your First HTML File

Every HTML page looks like this:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Website</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>
```

---

## What Do These Tags Mean?

| Tag | What it does |
|-----|-------------|
| `<!DOCTYPE html>` | Tells the browser this is HTML |
| `<html>` | The start of the page |
| `<head>` | Hidden info about the page |
| `<title>` | The name shown in the browser tab |
| `<body>` | Everything visible on the page |
| `<h1>` | A big heading |
| `<p>` | A paragraph |

---

## Try It Yourself

1. Open Notepad (or any text editor)
2. Copy the code above
3. Save it as `index.html`
4. Open the file in your browser

**You just built your first webpage!** 🎉
