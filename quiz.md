### Question 10: Good Practice

A) `<a href="https://google.com">click here</a>`

## Answer Key

### Answers with Explanations:

**1. Answer: B**  
HTML stands for **HyperText Markup Language**. It's the standard markup language for creating web pages. (See [Lesson 01](lessons/01-what-is-html.md))

**2. Answer: C**  
The `<body>` tag contains everything visible on the webpage. The `<head>` contains hidden information like the title. (See [Lesson 02](lessons/02-your-first-webpage.md))

**3. Answer: A**  
`<h1>` is the largest heading (main title), while `<h6>` is the smallest. There are 6 heading levels total. (See [Lesson 03](lessons/03-headings-and-paragraphs.md))

**4. Answer: C**  
The `href` attribute specifies the URL or file path for a link. `src` is used for images, not links. (See [Lesson 04](lessons/04-links-and-images.md))

**5. Answer: C**  
The correct syntax is `<img src="image.jpg" alt="description">`. Images don't have closing tags, and you need both `src` and `alt` attributes. (See [Lesson 04](lessons/04-links-and-images.md))

**6. Answer: B**  
Use `<ol>` (ordered list) when the order matters, like steps in a recipe or rankings. Use `<ul>` (unordered) when order doesn't matter. (See [Lesson 05](lessons/05-lists-and-tables.md))

**7. Answer: A**  
`<th>` stands for **table header** and is used for column or row titles (usually bolded). `<td>` is used for regular data cells. (See [Lesson 05](lessons/05-lists-and-tables.md))

**8. Answer: B**  
Use `<input type="password">` for password fields. The text is hidden from view for security. (See [Lesson 06](lessons/06-forms.md))

**9. Answer: C**  
GitHub Pages, Netlify, and Vercel all offer **free hosting** for beginners. They're perfect for learning without paying. (See [Lesson 07](lessons/07-publishing-your-website.md))

**10. Answer: C**  
Use descriptive, meaningful text like "Search with Google" instead of vague text like "click here". This is better for accessibility and SEO. (See [Lesson 04](lessons/04-links-and-images.md))
